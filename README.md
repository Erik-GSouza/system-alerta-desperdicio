# 🍽️ Sistema de Alerta de Desperdício

Sistema web desenvolvido para monitorar a validade de alimentos doados, auxiliando ONGs e restaurantes institucionais a reduzir o desperdício.

---

## 🚀 Funcionalidades

- 🔐 Autenticação de usuários
- 📦 Cadastro de alimentos
- ⏰ Monitoramento de validade
- 🚨 Alertas de vencimento
- 📊 Dashboard com indicadores

---
